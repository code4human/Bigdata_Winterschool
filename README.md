# Bigdata_Winterschool
Bigdata_Winterschool is an educational event held from December 20 to 22, 2019 by KRIHS(국토연구원) and Dacon. 
I participated in beginners class and studied about data visualization. 
The final hackathon was produced by python libraries : pandas, matplotlib, folium, seaborn. Jeju bus data was offered for anaylsis.
My team won a participation prize.
